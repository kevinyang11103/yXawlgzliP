# 基于Python的Web漏洞挖掘系统 python907

## 项目概述

本项目是一款基于B/S架构的Web漏洞扫描平台，用于自动化检测网站的安全漏洞。系统采用Python技术，搭配MySQL数据库和Django框架，致力于提供操作简便、界面友好的安全检测工具。

## 技术栈

- Python
- MySQL
- Django框架

## 功能模块

- **SQL注入漏洞扫描模块**：自动扫描目标网站潜在的SQL注入漏洞。
- **漏洞报告生成模块**：生成详细的漏洞报告，便于用户理解和修复。

## 系统角色

- 用户：输入目标URL，启动漏洞扫描，阅读漏洞报告。

## 运行环境

- Python环境
- MySQL数据库环境
- 适用于主流Web服务器的部署环境

## 部署步骤

2. 部署系统依赖库。
3. 配置系统参数。
